Change event with < input>tag.

Note:

DOM events
1. Event Object
- event type: change, submit, to-1 unload, open, play
canplay, pause, playing, progress, ended, resize,
scroll, toggle etc.
- properties: type, target, preventDefault()
2. MouseEvent Object
3.Keyboard event object
4. FocusEvent Object
5. ClipboardEvent Object
6. DragEvent Object

Change event work with:
< input /> - text, number, password, email, color, radio,
checkbox, search, time, date, datetime, week, month, url,
tel, file,
< select>,
< textarea>
